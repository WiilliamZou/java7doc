049/**
050 * This class represents an Internet Protocol (IP) address.
051 *
052 * <p> An IP address is either a 32-bit or 128-bit unsigned number
053 * used by IP, a lower-level protocol on which protocols like UDP and
054 * TCP are built. The IP address architecture is defined by <a
055 * href="http://www.ietf.org/rfc/rfc790.txt"><i>RFC&nbsp;790:
056 * Assigned Numbers</i></a>, <a
057 * href="http://www.ietf.org/rfc/rfc1918.txt"> <i>RFC&nbsp;1918:
058 * Address Allocation for Private Internets</i></a>, <a
059 * href="http://www.ietf.org/rfc/rfc2365.txt"><i>RFC&nbsp;2365:
060 * Administratively Scoped IP Multicast</i></a>, and <a
061 * href="http://www.ietf.org/rfc/rfc2373.txt"><i>RFC&nbsp;2373: IP
062 * Version 6 Addressing Architecture</i></a>. An instance of an
063 * InetAddress consists of an IP address and possibly its
064 * corresponding host name (depending on whether it is constructed
065 * with a host name or whether it has already done reverse host name
066 * resolution).
067 *
068 * <h4> Address types </h4>
069 *
070 * <blockquote><table cellspacing=2 summary="Description of unicast and multicast address types">
071 *   <tr><th valign=top><i>unicast</i></th>
072 *       <td>An identifier for a single interface. A packet sent to
073 *         a unicast address is delivered to the interface identified by
074 *         that address.
075 *
076 *         <p> The Unspecified Address -- Also called anylocal or wildcard
077 *         address. It must never be assigned to any node. It indicates the
078 *         absence of an address. One example of its use is as the target of
079 *         bind, which allows a server to accept a client connection on any
080 *         interface, in case the server host has multiple interfaces.
081 *
082 *         <p> The <i>unspecified</i> address must not be used as
083 *         the destination address of an IP packet.
084 *
085 *         <p> The <i>Loopback</i> Addresses -- This is the address
086 *         assigned to the loopback interface. Anything sent to this
087 *         IP address loops around and becomes IP input on the local
088 *         host. This address is often used when testing a
089 *         client.</td></tr>
090 *   <tr><th valign=top><i>multicast</i></th>
091 *       <td>An identifier for a set of interfaces (typically belonging
092 *         to different nodes). A packet sent to a multicast address is
093 *         delivered to all interfaces identified by that address.</td></tr>
094 * </table></blockquote>
095 *
096 * <h4> IP address scope </h4>
097 *
098 * <p> <i>Link-local</i> addresses are designed to be used for addressing
099 * on a single link for purposes such as auto-address configuration,
100 * neighbor discovery, or when no routers are present.
101 *
102 * <p> <i>Site-local</i> addresses are designed to be used for addressing
103 * inside of a site without the need for a global prefix.
104 *
105 * <p> <i>Global</i> addresses are unique across the internet.
106 *
107 * <h4> Textual representation of IP addresses </h4>
108 *
109 * The textual representation of an IP address is address family specific.
110 *
111 * <p>
112 *
113 * For IPv4 address format, please refer to <A
114 * HREF="Inet4Address.html#format">Inet4Address#format</A>; For IPv6
115 * address format, please refer to <A
116 * HREF="Inet6Address.html#format">Inet6Address#format</A>.
117 *
118 * <P>There is a <a href="doc-files/net-properties.html#Ipv4IPv6">couple of
119 * System Properties</a> affecting how IPv4 and IPv6 addresses are used.</P>
120 *
121 * <h4> Host Name Resolution </h4>
122 *
123 * Host name-to-IP address <i>resolution</i> is accomplished through
124 * the use of a combination of local machine configuration information
125 * and network naming services such as the Domain Name System (DNS)
126 * and Network Information Service(NIS). The particular naming
127 * services(s) being used is by default the local machine configured
128 * one. For any host name, its corresponding IP address is returned.
129 *
130 * <p> <i>Reverse name resolution</i> means that for any IP address,
131 * the host associated with the IP address is returned.
132 *
133 * <p> The InetAddress class provides methods to resolve host names to
134 * their IP addresses and vice versa.
135 *
136 * <h4> InetAddress Caching </h4>
137 *
138 * The InetAddress class has a cache to store successful as well as
139 * unsuccessful host name resolutions.
140 *
141 * <p> By default, when a security manager is installed, in order to
142 * protect against DNS spoofing attacks,
143 * the result of positive host name resolutions are
144 * cached forever. When a security manager is not installed, the default
145 * behavior is to cache entries for a finite (implementation dependent)
146 * period of time. The result of unsuccessful host
147 * name resolution is cached for a very short period of time (10
148 * seconds) to improve performance.
149 *
150 * <p> If the default behavior is not desired, then a Java security property
151 * can be set to a different Time-to-live (TTL) value for positive
152 * caching. Likewise, a system admin can configure a different
153 * negative caching TTL value when needed.
154 *
155 * <p> Two Java security properties control the TTL values used for
156 *  positive and negative host name resolution caching:
157 *
158 * <blockquote>
159 * <dl>
160 * <dt><b>networkaddress.cache.ttl</b></dt>
161 * <dd>Indicates the caching policy for successful name lookups from
162 * the name service. The value is specified as as integer to indicate
163 * the number of seconds to cache the successful lookup. The default
164 * setting is to cache for an implementation specific period of time.
165 * <p>
166 * A value of -1 indicates "cache forever".
167 * </dd>
168 * <p>
169 * <dt><b>networkaddress.cache.negative.ttl</b> (default: 10)</dt>
170 * <dd>Indicates the caching policy for un-successful name lookups
171 * from the name service. The value is specified as as integer to
172 * indicate the number of seconds to cache the failure for
173 * un-successful lookups.
174 * <p>
175 * A value of 0 indicates "never cache".
176 * A value of -1 indicates "cache forever".
177 * </dd>
178 * </dl>
179 * </blockquote>
180 *
181 * @author  Chris Warth
182 * @see     java.net.InetAddress#getByAddress(byte[])
183 * @see     java.net.InetAddress#getByAddress(java.lang.String, byte[])
184 * @see     java.net.InetAddress#getAllByName(java.lang.String)
185 * @see     java.net.InetAddress#getByName(java.lang.String)
186 * @see     java.net.InetAddress#getLocalHost()
187 * @since JDK1.0
188 */

437    /**
438     * Test whether that address is reachable. Best effort is made by the
439     * implementation to try to reach the host, but firewalls and server
440     * configuration may block requests resulting in a unreachable status
441     * while some specific ports may be accessible.
442     * A typical implementation will use ICMP ECHO REQUESTs if the
443     * privilege can be obtained, otherwise it will try to establish
444     * a TCP connection on port 7 (Echo) of the destination host.
445     * <p>
446     * The <code>network interface</code> and <code>ttl</code> parameters
447     * let the caller specify which network interface the test will go through
448     * and the maximum number of hops the packets should go through.
449     * A negative value for the <code>ttl</code> will result in an
450     * IllegalArgumentException being thrown.
451     * <p>
452     * The timeout value, in milliseconds, indicates the maximum amount of time
453     * the try should take. If the operation times out before getting an
454     * answer, the host is deemed unreachable. A negative value will result
455     * in an IllegalArgumentException being thrown.
456     *
457     * @param   netif   the NetworkInterface through which the
458     *                    test will be done, or null for any interface
459     * @param   ttl     the maximum numbers of hops to try or 0 for the
460     *                  default
461     * @param   timeout the time, in milliseconds, before the call aborts
462     * @throws  IllegalArgumentException if either <code>timeout</code>
463     *                          or <code>ttl</code> are negative.
464     * @return a <code>boolean</code>indicating if the address is reachable.
465     * @throws IOException if a network error occurs
466     * @since 1.5
467     */
468    public boolean isReachable(NetworkInterface netif, int ttl,
469                               int timeout) throws IOException {
470        if (ttl < 0)
471            throw new IllegalArgumentException("ttl can't be negative");
472        if (timeout < 0)
473            throw new IllegalArgumentException("timeout can't be negative");
474
475        return impl.isReachable(this, timeout, netif, ttl);
476    }

566    /**
567     * Returns the hostname for this address.
568     *
569     * <p>If there is a security manager, this method first
570     * calls its <code>checkConnect</code> method
571     * with the hostname and <code>-1</code>
572     * as its arguments to see if the calling code is allowed to know
573     * the hostname for this IP address, i.e., to connect to the host.
574     * If the operation is not allowed, it will return
575     * the textual representation of the IP address.
576     *
577     * @return  the host name for this IP address, or if the operation
578     *    is not allowed by the security check, the textual
579     *    representation of the IP address.
580     *
581     * @param check make security check if true
582     *
583     * @see SecurityManager#checkConnect
584     */
585    private static String getHostFromNameService(InetAddress addr, boolean check) {
586        String host = null;
587        for (NameService nameService : nameServices) {
588            try {
589                // first lookup the hostname
590                host = nameService.getHostByAddr(addr.getAddress());
591
592                /* check to see if calling code is allowed to know
593                 * the hostname for this IP address, ie, connect to the host
594                 */
595                if (check) {
596                    SecurityManager sec = System.getSecurityManager();
597                    if (sec != null) {
598                        sec.checkConnect(host, -1);
599                    }
600                }
601
602                /* now get all the IP addresses for this hostname,
603                 * and make sure one of them matches the original IP
604                 * address. We do this to try and prevent spoofing.
605                 */
606
607                InetAddress[] arr = InetAddress.getAllByName0(host, check);
608                boolean ok = false;
609
610                if(arr != null) {
611                    for(int i = 0; !ok && i < arr.length; i++) {
612                        ok = addr.equals(arr[i]);
613                    }
614                }
615
616                //XXX: if it looks a spoof just return the address?
617                if (!ok) {
618                    host = addr.getHostAddress();
619                    return host;
620                }
621
622                break;
623
624            } catch (SecurityException e) {
625                host = addr.getHostAddress();
626                break;
627            } catch (UnknownHostException e) {
628                host = addr.getHostAddress();
629                // let next provider resolve the hostname
630            }
631        }
632
633        return host;
634    }

717    /**
718     * Represents a cache entry
719     */
720    static final class CacheEntry {
721
722        CacheEntry(InetAddress[] addresses, long expiration) {
723            this.addresses = addresses;
724            this.expiration = expiration;
725        }
726
727        InetAddress[] addresses;
728        long expiration;
729    }

731    /**
732     * A cache that manages entries based on a policy specified
733     * at creation time.
734     */
735    static final class Cache {
736        private LinkedHashMap<String, CacheEntry> cache;
737        private Type type;
738
739        enum Type {Positive, Negative};
740
741        /**
742         * Create cache
743         */
744        public Cache(Type type) {
745            this.type = type;
746            cache = new LinkedHashMap<String, CacheEntry>();
747        }
748
749        private int getPolicy() {
750            if (type == Type.Positive) {
751                return InetAddressCachePolicy.get();
752            } else {
753                return InetAddressCachePolicy.getNegative();
754            }
755        }
756
757        /**
758         * Add an entry to the cache. If there's already an
759         * entry then for this host then the entry will be
760         * replaced.
761         */
762        public Cache put(String host, InetAddress[] addresses) {
763            int policy = getPolicy();
764            if (policy == InetAddressCachePolicy.NEVER) {
765                return this;
766            }
767
768            // purge any expired entries
769
770            if (policy != InetAddressCachePolicy.FOREVER) {
771
772                // As we iterate in insertion order we can
773                // terminate when a non-expired entry is found.
774                LinkedList<String> expired = new LinkedList<>();
775                long now = System.currentTimeMillis();
776                for (String key : cache.keySet()) {
777                    CacheEntry entry = cache.get(key);
778
779                    if (entry.expiration >= 0 && entry.expiration < now) {
780                        expired.add(key);
781                    } else {
782                        break;
783                    }
784                }
785
786                for (String key : expired) {
787                    cache.remove(key);
788                }
789            }
790
791            // create new entry and add it to the cache
792            // -- as a HashMap replaces existing entries we
793            //    don't need to explicitly check if there is
794            //    already an entry for this host.
795            long expiration;
796            if (policy == InetAddressCachePolicy.FOREVER) {
797                expiration = -1;
798            } else {
799                expiration = System.currentTimeMillis() + (policy * 1000);
800            }
801            CacheEntry entry = new CacheEntry(addresses, expiration);
802            cache.put(host, entry);
803            return this;
804        }
805
806        /**
807         * Query the cache for the specific host. If found then
808         * return its CacheEntry, or null if not found.
809         */
810        public CacheEntry get(String host) {
811            int policy = getPolicy();
812            if (policy == InetAddressCachePolicy.NEVER) {
813                return null;
814            }
815            CacheEntry entry = cache.get(host);
816
817            // check if entry has expired
818            if (entry != null && policy != InetAddressCachePolicy.FOREVER) {
819                if (entry.expiration >= 0 &&
820                    entry.expiration < System.currentTimeMillis()) {
821                    cache.remove(host);
822                    entry = null;
823                }
824            }
825
826            return entry;
827        }
828    }

973     * Creates an InetAddress based on the provided host name and IP address.
974     * No name service is checked for the validity of the address.
975     *
976     * <p> The host name can either be a machine name, such as
977     * "<code>java.sun.com</code>", or a textual representation of its IP
978     * address.
979     * <p> No validity checking is done on the host name either.
980     *
981     * <p> If addr specifies an IPv4 address an instance of Inet4Address
982     * will be returned; otherwise, an instance of Inet6Address
983     * will be returned.
984     *
985     * <p> IPv4 address byte array must be 4 bytes long and IPv6 byte array
986     * must be 16 bytes long
987     *
988     * @param host the specified host
989     * @param addr the raw IP address in network byte order
990     * @return  an InetAddress object created from the raw IP address.
991     * @exception  UnknownHostException  if IP address is of illegal length
992     * @since 1.4
993     */
994    public static InetAddress getByAddress(String host, byte[] addr)
995        throws UnknownHostException {
996        if (host != null && host.length() > 0 && host.charAt(0) == '[') {
997            if (host.charAt(host.length()-1) == ']') {
998                host = host.substring(1, host.length() -1);
999            }
1000        }
1001        if (addr != null) {
1002            if (addr.length == Inet4Address.INADDRSZ) {
1003                return new Inet4Address(host, addr);
1004            } else if (addr.length == Inet6Address.INADDRSZ) {
1005                byte[] newAddr
1006                    = IPAddressUtil.convertFromIPv4MappedAddress(addr);
1007                if (newAddr != null) {
1008                    return new Inet4Address(host, newAddr);
1009                } else {
1010                    return new Inet6Address(host, addr);
1011                }
1012            }
1013        }
1014        throw new UnknownHostException("addr is of illegal length");
1015    }

1101    private static InetAddress[] getAllByName(String host, InetAddress reqAddr)
1102        throws UnknownHostException {
1103
1104        if (host == null || host.length() == 0) {
1105            InetAddress[] ret = new InetAddress[1];
1106            ret[0] = impl.loopbackAddress();
1107            return ret;
1108        }
1109
1110        boolean ipv6Expected = false;
1111        if (host.charAt(0) == '[') {
1112            // This is supposed to be an IPv6 literal
1113            if (host.length() > 2 && host.charAt(host.length()-1) == ']') {
1114                host = host.substring(1, host.length() -1);
1115                ipv6Expected = true;
1116            } else {
1117                // This was supposed to be a IPv6 address, but it's not!
1118                throw new UnknownHostException(host + ": invalid IPv6 address");
1119            }
1120        }
1121
1122        // if host is an IP address, we won't do further lookup
1123        if (Character.digit(host.charAt(0), 16) != -1
1124            || (host.charAt(0) == ':')) {
1125            byte[] addr = null;
1126            int numericZone = -1;
1127            String ifname = null;
1128            // see if it is IPv4 address
1129            addr = IPAddressUtil.textToNumericFormatV4(host);
1130            if (addr == null) {
1131                // This is supposed to be an IPv6 literal
1132                // Check if a numeric or string zone id is present
1133                int pos;
1134                if ((pos=host.indexOf ("%")) != -1) {
1135                    numericZone = checkNumericZone (host);
1136                    if (numericZone == -1) { /* remainder of string must be an ifname */
1137                        ifname = host.substring (pos+1);
1138                    }
1139                }
1140                if ((addr = IPAddressUtil.textToNumericFormatV6(host)) == null && host.contains(":")) {
1141                    throw new UnknownHostException(host + ": invalid IPv6 address");
1142                }
1143            } else if (ipv6Expected) {
1144                // Means an IPv4 litteral between brackets!
1145                throw new UnknownHostException("["+host+"]");
1146            }
1147            InetAddress[] ret = new InetAddress[1];
1148            if(addr != null) {
1149                if (addr.length == Inet4Address.INADDRSZ) {
1150                    ret[0] = new Inet4Address(null, addr);
1151                } else {
1152                    if (ifname != null) {
1153                        ret[0] = new Inet6Address(null, addr, ifname);
1154                    } else {
1155                        ret[0] = new Inet6Address(null, addr, numericZone);
1156                    }
1157                }
1158                return ret;
1159            }
1160        } else if (ipv6Expected) {
1161            // We were expecting an IPv6 Litteral, but got something else
1162            throw new UnknownHostException("["+host+"]");
1163        }
1164        return getAllByName0(host, reqAddr, true);
1165    }

1183    /**
1184     * check if the literal address string has %nn appended
1185     * returns -1 if not, or the numeric value otherwise.
1186     *
1187     * %nn may also be a string that represents the displayName of
1188     * a currently available NetworkInterface.
1189     */
1190    private static int checkNumericZone (String s) throws UnknownHostException {
1191        int percent = s.indexOf ('%');
1192        int slen = s.length();
1193        int digit, zone=0;
1194        if (percent == -1) {
1195            return -1;
1196        }
1197        for (int i=percent+1; i<slen; i++) {
1198            char c = s.charAt(i);
1199            if (c == ']') {
1200                if (i == percent+1) {
1201                    /* empty per-cent field */
1202                    return -1;
1203                }
1204                break;
1205            }
1206            if ((digit = Character.digit (c, 10)) < 0) {
1207                return -1;
1208            }
1209            zone = (zone * 10) + digit;
1210        }
1211        return zone;
1212    }

1257    private static InetAddress[] getAddressesFromNameService(String host, InetAddress reqAddr)
1258        throws UnknownHostException
1259    {
1260        InetAddress[] addresses = null;
1261        boolean success = false;
1262        UnknownHostException ex = null;
1263
1264        // Check whether the host is in the lookupTable.
1265        // 1) If the host isn't in the lookupTable when
1266        //    checkLookupTable() is called, checkLookupTable()
1267        //    would add the host in the lookupTable and
1268        //    return null. So we will do the lookup.
1269        // 2) If the host is in the lookupTable when
1270        //    checkLookupTable() is called, the current thread
1271        //    would be blocked until the host is removed
1272        //    from the lookupTable. Then this thread
1273        //    should try to look up the addressCache.
1274        //     i) if it found the addresses in the
1275        //        addressCache, checkLookupTable()  would
1276        //        return the addresses.
1277        //     ii) if it didn't find the addresses in the
1278        //         addressCache for any reason,
1279        //         it should add the host in the
1280        //         lookupTable and return null so the
1281        //         following code would do  a lookup itself.
1282        if ((addresses = checkLookupTable(host)) == null) {
1283            try {
1284                // This is the first thread which looks up the addresses
1285                // this host or the cache entry for this host has been
1286                // expired so this thread should do the lookup.
1287                for (NameService nameService : nameServices) {
1288                    try {
1289                        /*
1290                         * Do not put the call to lookup() inside the
1291                         * constructor.  if you do you will still be
1292                         * allocating space when the lookup fails.
1293                         */
1294
1295                        addresses = nameService.lookupAllHostAddr(host);
1296                        success = true;
1297                        break;
1298                    } catch (UnknownHostException uhe) {
1299                        if (host.equalsIgnoreCase("localhost")) {
1300                            InetAddress[] local = new InetAddress[] { impl.loopbackAddress() };
1301                            addresses = local;
1302                            success = true;
1303                            break;
1304                        }
1305                        else {
1306                            addresses = unknown_array;
1307                            success = false;
1308                            ex = uhe;
1309                        }
1310                    }
1311                }
1312
1313                // More to do?
1314                if (reqAddr != null && addresses.length > 1 && !addresses[0].equals(reqAddr)) {
1315                    // Find it?
1316                    int i = 1;
1317                    for (; i < addresses.length; i++) {
1318                        if (addresses[i].equals(reqAddr)) {
1319                            break;
1320                        }
1321                    }
1322                    // Rotate
1323                    if (i < addresses.length) {
1324                        InetAddress tmp, tmp2 = reqAddr;
1325                        for (int j = 0; j < i; j++) {
1326                            tmp = addresses[j];
1327                            addresses[j] = tmp2;
1328                            tmp2 = tmp;
1329                        }
1330                        addresses[i] = tmp2;
1331                    }
1332                }
1333                // Cache the address.
1334                cacheAddresses(host, addresses, success);
1335
1336                if (!success && ex != null)
1337                    throw ex;
1338
1339            } finally {
1340                // Delete host from the lookupTable and notify
1341                // all threads waiting on the lookupTable monitor.
1342                updateLookupTable(host);
1343            }
1344        }
1345
1346        return addresses;
1347    }

1350    private static InetAddress[] checkLookupTable(String host) {
1351        synchronized (lookupTable) {
1352            // If the host isn't in the lookupTable, add it in the
1353            // lookuptable and return null. The caller should do
1354            // the lookup.
1355            if (lookupTable.containsKey(host) == false) {
1356                lookupTable.put(host, null);
1357                return null;
1358            }
1359
1360            // If the host is in the lookupTable, it means that another
1361            // thread is trying to look up the addresses of this host.
1362            // This thread should wait.
1363            while (lookupTable.containsKey(host)) {
1364                try {
1365                    lookupTable.wait();
1366                } catch (InterruptedException e) {
1367                }
1368            }
1369        }
1370
1371        // The other thread has finished looking up the addresses of
1372        // the host. This thread should retry to get the addresses
1373        // from the addressCache. If it doesn't get the addresses from
1374        // the cache, it will try to look up the addresses itself.
1375        InetAddress[] addresses = getCachedAddresses(host);
1376        if (addresses == null) {
1377            synchronized (lookupTable) {
1378                lookupTable.put(host, null);
1379                return null;
1380            }
1381        }
1382
1383        return addresses;
1384    }

1386    private static void updateLookupTable(String host) {
1387        synchronized (lookupTable) {
1388            lookupTable.remove(host);
1389            lookupTable.notifyAll();
1390        }
1391    }

1396     * byte of the address is in <code>getAddress()[0]</code>.
1397     *
1398     * <p> This method doesn't block, i.e. no reverse name service lookup
1399     * is performed.
1400     *
1401     * <p> IPv4 address byte array must be 4 bytes long and IPv6 byte array
1402     * must be 16 bytes long
1403     *
1404     * @param addr the raw IP address in network byte order
1405     * @return  an InetAddress object created from the raw IP address.
1406     * @exception  UnknownHostException  if IP address is of illegal length
1407     * @since 1.4
1408     */
1409    public static InetAddress getByAddress(byte[] addr)
1410        throws UnknownHostException {
1411        return getByAddress(null, addr);
1412    }

1414    private static InetAddress cachedLocalHost = null;
1415    private static long cacheTime = 0;
1416    private static final long maxCacheTime = 5000L;
1417    private static final Object cacheLock = new Object();
1418
1419    /**
1420     * Returns the address of the local host. This is achieved by retrieving
1421     * the name of the host from the system, then resolving that name into
1422     * an <code>InetAddress</code>.
1423     *
1424     * <P>Note: The resolved address may be cached for a short period of time.
1425     * </P>
1426     *
1427     * <p>If there is a security manager, its
1428     * <code>checkConnect</code> method is called
1429     * with the local host name and <code>-1</code>
1430     * as its arguments to see if the operation is allowed.
1431     * If the operation is not allowed, an InetAddress representing
1432     * the loopback address is returned.
1433     *
1434     * @return     the address of the local host.
1435     *
1436     * @exception  UnknownHostException  if the local host name could not
1437     *             be resolved into an address.
1438     *
1439     * @see SecurityManager#checkConnect
1440     * @see java.net.InetAddress#getByName(java.lang.String)
1441     */
1442    public static InetAddress getLocalHost() throws UnknownHostException {
1443
1444        SecurityManager security = System.getSecurityManager();
1445        try {
1446            String local = impl.getLocalHostName();
1447
1448            if (security != null) {
1449                security.checkConnect(local, -1);
1450            }
1451
1452            if (local.equals("localhost")) {
1453                return impl.loopbackAddress();
1454            }
1455
1456            InetAddress ret = null;
1457            synchronized (cacheLock) {
1458                long now = System.currentTimeMillis();
1459                if (cachedLocalHost != null) {
1460                    if ((now - cacheTime) < maxCacheTime) // Less than 5s old?
1461                        ret = cachedLocalHost;
1462                    else
1463                        cachedLocalHost = null;
1464                }
1465
1466                // we are calling getAddressesFromNameService directly
1467                // to avoid getting localHost from cache
1468                if (ret == null) {
1469                    InetAddress[] localAddrs;
1470                    try {
1471                        localAddrs =
1472                            InetAddress.getAddressesFromNameService(local, null);
1473                    } catch (UnknownHostException uhe) {
1474                        // Rethrow with a more informative error message.
1475                        UnknownHostException uhe2 =
1476                            new UnknownHostException(local + ": " +
1477                                                     uhe.getMessage());
1478                        uhe2.initCause(uhe);
1479                        throw uhe2;
1480                    }
1481                    cachedLocalHost = localAddrs[0];
1482                    cacheTime = now;
1483                    ret = localAddrs[0];
1484                }
1485            }
1486            return ret;
1487        } catch (java.lang.SecurityException e) {
1488            return impl.loopbackAddress();
1489        }
1490    }

1506    /*
1507     * Load and instantiate an underlying impl class
1508     */
1509    static InetAddressImpl loadImpl(String implName) {
1510        Object impl = null;
1511
1512        /*
1513         * Property "impl.prefix" will be prepended to the classname
1514         * of the implementation object we instantiate, to which we
1515         * delegate the real work (like native methods).  This
1516         * property can vary across implementations of the java.
1517         * classes.  The default is an empty String "".
1518         */
1519        String prefix = AccessController.doPrivileged(
1520                      new GetPropertyAction("impl.prefix", ""));
1521        try {
1522            impl = Class.forName("java.net." + prefix + implName).newInstance();
1523        } catch (ClassNotFoundException e) {
1524            System.err.println("Class not found: java.net." + prefix +
1525                               implName + ":\ncheck impl.prefix property " +
1526                               "in your properties file.");
1527        } catch (InstantiationException e) {
1528            System.err.println("Could not instantiate: java.net." + prefix +
1529                               implName + ":\ncheck impl.prefix property " +
1530                               "in your properties file.");
1531        } catch (IllegalAccessException e) {
1532            System.err.println("Cannot access class: java.net." + prefix +
1533                               implName + ":\ncheck impl.prefix property " +
1534                               "in your properties file.");
1535        }
1536
1537        if (impl == null) {
1538            try {
1539                impl = Class.forName(implName).newInstance();
1540            } catch (Exception e) {
1541                throw new Error("System property impl.prefix incorrect");
1542            }
1543        }
1544
1545        return (InetAddressImpl) impl;
1546    }

1571                         IOException, ClassNotFoundException {

1574        }

1577        int address= gf.get("address", 0);
1578        int family= gf.get("family", 0);
1579        InetAddressHolder h = new InetAddressHolder(host, address, family);
1580        UNSAFE.putObject(this, FIELDS_OFFSET, h);

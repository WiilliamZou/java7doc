043/**
044 * {@code Proxy} provides static methods for creating dynamic proxy
045 * classes and instances, and it is also the superclass of all
046 * dynamic proxy classes created by those methods.
047 *
048 * <p>To create a proxy for some interface {@code Foo}:
049 * <pre>
050 *     InvocationHandler handler = new MyInvocationHandler(...);
051 *     Class proxyClass = Proxy.getProxyClass(
052 *         Foo.class.getClassLoader(), new Class[] { Foo.class });
053 *     Foo f = (Foo) proxyClass.
054 *         getConstructor(new Class[] { InvocationHandler.class }).
055 *         newInstance(new Object[] { handler });
056 * </pre>
057 * or more simply:
058 * <pre>
059 *     Foo f = (Foo) Proxy.newProxyInstance(Foo.class.getClassLoader(),
060 *                                          new Class[] { Foo.class },
061 *                                          handler);
062 * </pre>
063 *
064 * <p>A <i>dynamic proxy class</i> (simply referred to as a <i>proxy
065 * class</i> below) is a class that implements a list of interfaces
066 * specified at runtime when the class is created, with behavior as
067 * described below.
068 *
069 * A <i>proxy interface</i> is such an interface that is implemented
070 * by a proxy class.
071 *
072 * A <i>proxy instance</i> is an instance of a proxy class.
073 *
074 * Each proxy instance has an associated <i>invocation handler</i>
075 * object, which implements the interface {@link InvocationHandler}.
076 * A method invocation on a proxy instance through one of its proxy
077 * interfaces will be dispatched to the {@link InvocationHandler#invoke
078 * invoke} method of the instance's invocation handler, passing the proxy
079 * instance, a {@code java.lang.reflect.Method} object identifying
080 * the method that was invoked, and an array of type {@code Object}
081 * containing the arguments.  The invocation handler processes the
082 * encoded method invocation as appropriate and the result that it
083 * returns will be returned as the result of the method invocation on
084 * the proxy instance.
085 *
086 * <p>A proxy class has the following properties:
087 *
088 * <ul>
089 * <li>Proxy classes are public, final, and not abstract.
090 *
091 * <li>The unqualified name of a proxy class is unspecified.  The space
092 * of class names that begin with the string {@code "$Proxy"}
093 * should be, however, reserved for proxy classes.
094 *
095 * <li>A proxy class extends {@code java.lang.reflect.Proxy}.
096 *
097 * <li>A proxy class implements exactly the interfaces specified at its
098 * creation, in the same order.
099 *
100 * <li>If a proxy class implements a non-public interface, then it will
101 * be defined in the same package as that interface.  Otherwise, the
102 * package of a proxy class is also unspecified.  Note that package
103 * sealing will not prevent a proxy class from being successfully defined
104 * in a particular package at runtime, and neither will classes already
105 * defined by the same class loader and the same package with particular
106 * signers.
107 *
108 * <li>Since a proxy class implements all of the interfaces specified at
109 * its creation, invoking {@code getInterfaces} on its
110 * {@code Class} object will return an array containing the same
111 * list of interfaces (in the order specified at its creation), invoking
112 * {@code getMethods} on its {@code Class} object will return
113 * an array of {@code Method} objects that include all of the
114 * methods in those interfaces, and invoking {@code getMethod} will
115 * find methods in the proxy interfaces as would be expected.
116 *
117 * <li>The {@link Proxy#isProxyClass Proxy.isProxyClass} method will
118 * return true if it is passed a proxy class-- a class returned by
119 * {@code Proxy.getProxyClass} or the class of an object returned by
120 * {@code Proxy.newProxyInstance}-- and false otherwise.
121 *
122 * <li>The {@code java.security.ProtectionDomain} of a proxy class
123 * is the same as that of system classes loaded by the bootstrap class
124 * loader, such as {@code java.lang.Object}, because the code for a
125 * proxy class is generated by trusted system code.  This protection
126 * domain will typically be granted
127 * {@code java.security.AllPermission}.
128 *
129 * <li>Each proxy class has one public constructor that takes one argument,
130 * an implementation of the interface {@link InvocationHandler}, to set
131 * the invocation handler for a proxy instance.  Rather than having to use
132 * the reflection API to access the public constructor, a proxy instance
133 * can be also be created by calling the {@link Proxy#newProxyInstance
134 * Proxy.newProxyInstance} method, which combines the actions of calling
135 * {@link Proxy#getProxyClass Proxy.getProxyClass} with invoking the
136 * constructor with an invocation handler.
137 * </ul>
138 *
139 * <p>A proxy instance has the following properties:
140 *
141 * <ul>
142 * <li>Given a proxy instance {@code proxy} and one of the
143 * interfaces implemented by its proxy class {@code Foo}, the
144 * following expression will return true:
145 * <pre>
146 *     {@code proxy instanceof Foo}
147 * </pre>
148 * and the following cast operation will succeed (rather than throwing
149 * a {@code ClassCastException}):
150 * <pre>
151 *     {@code (Foo) proxy}
152 * </pre>
153 *
154 * <li>Each proxy instance has an associated invocation handler, the one
155 * that was passed to its constructor.  The static
156 * {@link Proxy#getInvocationHandler Proxy.getInvocationHandler} method
157 * will return the invocation handler associated with the proxy instance
158 * passed as its argument.
159 *
160 * <li>An interface method invocation on a proxy instance will be
161 * encoded and dispatched to the invocation handler's {@link
162 * InvocationHandler#invoke invoke} method as described in the
163 * documentation for that method.
164 *
165 * <li>An invocation of the {@code hashCode},
166 * {@code equals}, or {@code toString} methods declared in
167 * {@code java.lang.Object} on a proxy instance will be encoded and
168 * dispatched to the invocation handler's {@code invoke} method in
169 * the same manner as interface method invocations are encoded and
170 * dispatched, as described above.  The declaring class of the
171 * {@code Method} object passed to {@code invoke} will be
172 * {@code java.lang.Object}.  Other public methods of a proxy
173 * instance inherited from {@code java.lang.Object} are not
174 * overridden by a proxy class, so invocations of those methods behave
175 * like they do for instances of {@code java.lang.Object}.
176 * </ul>
177 *
178 * <h3>Methods Duplicated in Multiple Proxy Interfaces</h3>
179 *
180 * <p>When two or more interfaces of a proxy class contain a method with
181 * the same name and parameter signature, the order of the proxy class's
182 * interfaces becomes significant.  When such a <i>duplicate method</i>
183 * is invoked on a proxy instance, the {@code Method} object passed
184 * to the invocation handler will not necessarily be the one whose
185 * declaring class is assignable from the reference type of the interface
186 * that the proxy's method was invoked through.  This limitation exists
187 * because the corresponding method implementation in the generated proxy
188 * class cannot determine which interface it was invoked through.
189 * Therefore, when a duplicate method is invoked on a proxy instance,
190 * the {@code Method} object for the method in the foremost interface
191 * that contains the method (either directly or inherited through a
192 * superinterface) in the proxy class's list of interfaces is passed to
193 * the invocation handler's {@code invoke} method, regardless of the
194 * reference type through which the method invocation occurred.
195 *
196 * <p>If a proxy interface contains a method with the same name and
197 * parameter signature as the {@code hashCode}, {@code equals},
198 * or {@code toString} methods of {@code java.lang.Object},
199 * when such a method is invoked on a proxy instance, the
200 * {@code Method} object passed to the invocation handler will have
201 * {@code java.lang.Object} as its declaring class.  In other words,
202 * the public, non-final methods of {@code java.lang.Object}
203 * logically precede all of the proxy interfaces for the determination of
204 * which {@code Method} object to pass to the invocation handler.
205 *
206 * <p>Note also that when a duplicate method is dispatched to an
207 * invocation handler, the {@code invoke} method may only throw
208 * checked exception types that are assignable to one of the exception
209 * types in the {@code throws} clause of the method in <i>all</i> of
210 * the proxy interfaces that it can be invoked through.  If the
211 * {@code invoke} method throws a checked exception that is not
212 * assignable to any of the exception types declared by the method in one
213 * of the proxy interfaces that it can be invoked through, then an
214 * unchecked {@code UndeclaredThrowableException} will be thrown by
215 * the invocation on the proxy instance.  This restriction means that not
216 * all of the exception types returned by invoking
217 * {@code getExceptionTypes} on the {@code Method} object
218 * passed to the {@code invoke} method can necessarily be thrown
219 * successfully by the {@code invoke} method.
220 *
221 * @author      Peter Jones
222 * @see         InvocationHandler
223 * @since       1.3
224 */
225public class Proxy implements java.io.Serializable {

263    private static class ProxyAccessHelper {
264        // The permission is implementation specific.
265        static final Permission PROXY_PERMISSION =
266            new ReflectPermission("proxyConstructorNewInstance");
267        // These system properties are defined to provide a short-term
268        // workaround if customers need to disable the new security checks.
269        static final boolean allowNewInstance;
270        static final boolean allowNullLoader;
271        static {
272            allowNewInstance = getBooleanProperty("sun.reflect.proxy.allowsNewInstance");
273            allowNullLoader = getBooleanProperty("sun.reflect.proxy.allowsNullLoader");
274        }
275
276        private static boolean getBooleanProperty(final String key) {
277            String s = AccessController.doPrivileged(new PrivilegedAction<String>() {
278                public String run() {
279                    return System.getProperty(key);
280                }
281            });
282            return Boolean.valueOf(s);
283        }
284
285        static boolean needsNewInstanceCheck(Class<?> proxyClass) {
286            if (!Proxy.isProxyClass(proxyClass) || allowNewInstance) {
287                return false;
288            }
289
290            if (ReflectUtil.isNonPublicProxyClass(proxyClass)) {
291                for (Class<?> intf : proxyClass.getInterfaces()) {
292                    if (!Modifier.isPublic(intf.getModifiers())) {
293                        return true;
294                    }
295                }
296            }
297            return false;
298        }
299    }

301    /*
302     * Access check on a proxy class that implements any non-public interface.
303     *
304     * @throws  SecurityException if a security manager exists, and
305     *          the caller does not have the permission.
306     */
307    private void doNewInstanceCheck() {
308        SecurityManager sm = System.getSecurityManager();
309        Class<?> proxyClass = this.getClass();
310        if (sm != null && ProxyAccessHelper.needsNewInstanceCheck(proxyClass)) {
311            try {
312                sm.checkPermission(ProxyAccessHelper.PROXY_PERMISSION);
313            } catch (SecurityException e) {
314                throw new SecurityException("Not allowed to construct a Proxy "
315                        + "instance that implements a non-public interface", e);
316            }
317        }
318    }

320    /**
321     * Returns the {@code java.lang.Class} object for a proxy class
322     * given a class loader and an array of interfaces.  The proxy class
323     * will be defined by the specified class loader and will implement
324     * all of the supplied interfaces.  If a proxy class for the same
325     * permutation of interfaces has already been defined by the class
326     * loader, then the existing proxy class will be returned; otherwise,
327     * a proxy class for those interfaces will be generated dynamically
328     * and defined by the class loader.
329     *
330     * <p>There are several restrictions on the parameters that may be
331     * passed to {@code Proxy.getProxyClass}:
332     *
333     * <ul>
334     * <li>All of the {@code Class} objects in the
335     * {@code interfaces} array must represent interfaces, not
336     * classes or primitive types.
337     *
338     * <li>No two elements in the {@code interfaces} array may
339     * refer to identical {@code Class} objects.
340     *
341     * <li>All of the interface types must be visible by name through the
342     * specified class loader.  In other words, for class loader
343     * {@code cl} and every interface {@code i}, the following
344     * expression must be true:
345     * <pre>
346     *     Class.forName(i.getName(), false, cl) == i
347     * </pre>
348     *
349     * <li>All non-public interfaces must be in the same package;
350     * otherwise, it would not be possible for the proxy class to
351     * implement all of the interfaces, regardless of what package it is
352     * defined in.
353     *
354     * <li>For any set of member methods of the specified interfaces
355     * that have the same signature:
356     * <ul>
357     * <li>If the return type of any of the methods is a primitive
358     * type or void, then all of the methods must have that same
359     * return type.
360     * <li>Otherwise, one of the methods must have a return type that
361     * is assignable to all of the return types of the rest of the
362     * methods.
363     * </ul>
364     *
365     * <li>The resulting proxy class must not exceed any limits imposed
366     * on classes by the virtual machine.  For example, the VM may limit
367     * the number of interfaces that a class may implement to 65535; in
368     * that case, the size of the {@code interfaces} array must not
369     * exceed 65535.
370     * </ul>
371     *
372     * <p>If any of these restrictions are violated,
373     * {@code Proxy.getProxyClass} will throw an
374     * {@code IllegalArgumentException}.  If the {@code interfaces}
375     * array argument or any of its elements are {@code null}, a
376     * {@code NullPointerException} will be thrown.
377     *
378     * <p>Note that the order of the specified proxy interfaces is
379     * significant: two requests for a proxy class with the same combination
380     * of interfaces but in a different order will result in two distinct
381     * proxy classes.
382     *
383     * @param   loader the class loader to define the proxy class
384     * @param   interfaces the list of interfaces for the proxy class
385     *          to implement
386     * @return  a proxy class that is defined in the specified class loader
387     *          and that implements the specified interfaces
388     * @throws  IllegalArgumentException if any of the restrictions on the
389     *          parameters that may be passed to {@code getProxyClass}
390     *          are violated
391     * @throws  NullPointerException if the {@code interfaces} array
392     *          argument or any of its elements are {@code null}
393     */
394    @CallerSensitive
395    public static Class<?> getProxyClass(ClassLoader loader,
396                                         Class<?>... interfaces)
397        throws IllegalArgumentException
398    {
399        final Class<?>[] intfs = interfaces.clone();
400        final SecurityManager sm = System.getSecurityManager();
401        if (sm != null) {
402            checkProxyAccess(Reflection.getCallerClass(), loader, intfs);
403        }
404
405        return getProxyClass0(loader, intfs);
406    }

408    /*
409     * Check permissions required to create a Proxy class.
410     *
411     * To define a proxy class, it performs the access checks as in
412     * Class.forName (VM will invoke ClassLoader.checkPackageAccess):
413     * 1. "getClassLoader" permission check if loader == null
414     * 2. checkPackageAccess on the interfaces it implements
415     *
416     * To get a constructor and new instance of a proxy class, it performs
417     * the package access check on the interfaces it implements
418     * as in Class.getConstructor.
419     *
420     * If an interface is non-public, the proxy class must be defined by
421     * the defining loader of the interface.  If the caller's class loader
422     * is not the same as the defining loader of the interface, the VM
423     * will throw IllegalAccessError when the generated proxy class is
424     * being defined via the defineClass0 method.
425     */
426    private static void checkProxyAccess(Class<?> caller,
427                                         ClassLoader loader,
428                                         Class<?>... interfaces)
429    {
430        SecurityManager sm = System.getSecurityManager();
431        if (sm != null) {
432            ClassLoader ccl = caller.getClassLoader();
433            if (loader == null && ccl != null) {
434                if (!ProxyAccessHelper.allowNullLoader) {
435                    sm.checkPermission(SecurityConstants.GET_CLASSLOADER_PERMISSION);
436                }
437            }
438            ReflectUtil.checkProxyPackageAccess(ccl, interfaces);
439        }
440    }
441
442    /**
443     * Generate a proxy class.  Must call the checkProxyAccess method
444     * to perform permission checks before calling this.
445     */
446    private static Class<?> getProxyClass0(ClassLoader loader,
447                                           Class<?>... interfaces) {
448        if (interfaces.length > 65535) {
449            throw new IllegalArgumentException("interface limit exceeded");
450        }
451
452        // If the proxy class defined by the given loader implementing
453        // the given interfaces exists, this will simply return the cached copy;
454        // otherwise, it will create the proxy class via the ProxyClassFactory
455        return proxyClassCache.get(loader, interfaces);
456    }

468    /*
469     * a key used for proxy class with 1 implemented interface
470     */
471    private static final class Key1 extends WeakReference<Class<?>> {
472        private final int hash;
473
474        Key1(Class<?> intf) {
475            super(intf);
476            this.hash = intf.hashCode();
477        }
478
479        @Override
480        public int hashCode() {
481            return hash;
482        }
483
484        @Override
485        public boolean equals(Object obj) {
486            Class<?> intf;
487            return this == obj ||
488                   obj != null &&
489                   obj.getClass() == Key1.class &&
490                   (intf = get()) != null &&
491                   intf == ((Key1) obj).get();
492        }
493    }

495    /*
496     * a key used for proxy class with 2 implemented interfaces
497     */
498    private static final class Key2 extends WeakReference<Class<?>> {
499        private final int hash;
500        private final WeakReference<Class<?>> ref2;
501
502        Key2(Class<?> intf1, Class<?> intf2) {
503            super(intf1);
504            hash = 31 * intf1.hashCode() + intf2.hashCode();
505            ref2 = new WeakReference<Class<?>>(intf2);
506        }
507
508        @Override
509        public int hashCode() {
510            return hash;
511        }
512
513        @Override
514        public boolean equals(Object obj) {
515            Class<?> intf1, intf2;
516            return this == obj ||
517                   obj != null &&
518                   obj.getClass() == Key2.class &&
519                   (intf1 = get()) != null &&
520                   intf1 == ((Key2) obj).get() &&
521                   (intf2 = ref2.get()) != null &&
522                   intf2 == ((Key2) obj).ref2.get();
523        }
524    }

526    /*
527     * a key used for proxy class with any number of implemented interfaces
528     * (used here for 3 or more only)
529     */
530    private static final class KeyX {
531        private final int hash;
532        private final WeakReference<Class<?>>[] refs;
533
534        KeyX(Class<?>[] interfaces) {
535            hash = Arrays.hashCode(interfaces);
536            refs = new WeakReference[interfaces.length];
537            for (int i = 0; i < interfaces.length; i++) {
538                refs[i] = new WeakReference(interfaces[i]);
539            }
540        }
541
542        @Override
543        public int hashCode() {
544            return hash;
545        }
546
547        @Override
548        public boolean equals(Object obj) {
549            return this == obj ||
550                   obj != null &&
551                   obj.getClass() == KeyX.class &&
552                   equals(refs, ((KeyX) obj).refs);
553        }
554
555        private static boolean equals(WeakReference<Class<?>>[] refs1,
556                                      WeakReference<Class<?>>[] refs2) {
557            if (refs1.length != refs2.length) {
558                return false;
559            }
560            for (int i = 0; i < refs1.length; i++) {
561                Class<?> intf = refs1[i].get();
562                if (intf == null || intf != refs2[i].get()) {
563                    return false;
564                }
565            }
566            return true;
567        }
568    }

570    /**
571     * A function that maps an array of interfaces to an optimal key where
572     * Class objects representing interfaces are weakly referenced.
573     */
574    private static final class KeyFactory
575        implements BiFunction<ClassLoader, Class<?>[], Object>
576    {
577        @Override
578        public Object apply(ClassLoader classLoader, Class<?>[] interfaces) {
579            switch (interfaces.length) {
580                case 1: return new Key1(interfaces[0]); // the most frequent
581                case 2: return new Key2(interfaces[0], interfaces[1]);
582                case 0: return key0;
583                default: return new KeyX(interfaces);
584            }
585        }
586    }

588    /**
589     * A factory function that generates, defines and returns the proxy class given
590     * the ClassLoader and array of interfaces.
591     */
592    private static final class ProxyClassFactory
593        implements BiFunction<ClassLoader, Class<?>[], Class<?>>
594    {
595        // prefix for all proxy class names
596        private static final String proxyClassNamePrefix = "$Proxy";
597
598        // next number to use for generation of unique proxy class names
599        private static final AtomicLong nextUniqueNumber = new AtomicLong();
600
601        @Override
602        public Class<?> apply(ClassLoader loader, Class<?>[] interfaces) {
603
604            Map<Class<?>, Boolean> interfaceSet = new IdentityHashMap<>(interfaces.length);
605            for (Class<?> intf : interfaces) {
606                /*
607                 * Verify that the class loader resolves the name of this
608                 * interface to the same Class object.
609                 */
610                Class<?> interfaceClass = null;
611                try {
612                    interfaceClass = Class.forName(intf.getName(), false, loader);
613                } catch (ClassNotFoundException e) {
614                }
615                if (interfaceClass != intf) {
616                    throw new IllegalArgumentException(
617                        intf + " is not visible from class loader");
618                }
619                /*
620                 * Verify that the Class object actually represents an
621                 * interface.
622                 */
623                if (!interfaceClass.isInterface()) {
624                    throw new IllegalArgumentException(
625                        interfaceClass.getName() + " is not an interface");
626                }
627                /*
628                 * Verify that this interface is not a duplicate.
629                 */
630                if (interfaceSet.put(interfaceClass, Boolean.TRUE) != null) {
631                    throw new IllegalArgumentException(
632                        "repeated interface: " + interfaceClass.getName());
633                }
634            }
635
636            String proxyPkg = null;     // package to define proxy class in
637
638            /*
639             * Record the package of a non-public proxy interface so that the
640             * proxy class will be defined in the same package.  Verify that
641             * all non-public proxy interfaces are in the same package.
642             */
643            for (Class<?> intf : interfaces) {
644                int flags = intf.getModifiers();
645                if (!Modifier.isPublic(flags)) {
646                    String name = intf.getName();
647                    int n = name.lastIndexOf('.');
648                    String pkg = ((n == -1) ? "" : name.substring(0, n + 1));
649                    if (proxyPkg == null) {
650                        proxyPkg = pkg;
651                    } else if (!pkg.equals(proxyPkg)) {
652                        throw new IllegalArgumentException(
653                            "non-public interfaces from different packages");
654                    }
655                }
656            }
657
658            if (proxyPkg == null) {
659                // if no non-public proxy interfaces, use com.sun.proxy package
660                proxyPkg = ReflectUtil.PROXY_PACKAGE + ".";
661            }
662
663            /*
664             * Choose a name for the proxy class to generate.
665             */
666            long num = nextUniqueNumber.getAndIncrement();
667            String proxyName = proxyPkg + proxyClassNamePrefix + num;
668
669            /*
670             * Generate the specified proxy class.
671             */
672            byte[] proxyClassFile = ProxyGenerator.generateProxyClass(
673                proxyName, interfaces);
674            try {
675                return defineClass0(loader, proxyName,
676                                    proxyClassFile, 0, proxyClassFile.length);
677            } catch (ClassFormatError e) {
678                /*
679                 * A ClassFormatError here means that (barring bugs in the
680                 * proxy class generation code) there was some other
681                 * invalid aspect of the arguments supplied to the proxy
682                 * class creation (such as virtual machine limitations
683                 * exceeded).
684                 */
685                throw new IllegalArgumentException(e.toString());
686            }
687        }
688    }

690    /**
691     * Returns an instance of a proxy class for the specified interfaces
692     * that dispatches method invocations to the specified invocation
693     * handler.  This method is equivalent to:
694     * <pre>
695     *     Proxy.getProxyClass(loader, interfaces).
696     *         getConstructor(new Class[] { InvocationHandler.class }).
697     *         newInstance(new Object[] { handler });
698     * </pre>
699     *
700     * <p>{@code Proxy.newProxyInstance} throws
701     * {@code IllegalArgumentException} for the same reasons that
702     * {@code Proxy.getProxyClass} does.
703     *
704     * @param   loader the class loader to define the proxy class
705     * @param   interfaces the list of interfaces for the proxy class
706     *          to implement
707     * @param   h the invocation handler to dispatch method invocations to
708     * @return  a proxy instance with the specified invocation handler of a
709     *          proxy class that is defined by the specified class loader
710     *          and that implements the specified interfaces
711     * @throws  IllegalArgumentException if any of the restrictions on the
712     *          parameters that may be passed to {@code getProxyClass}
713     *          are violated
714     * @throws  NullPointerException if the {@code interfaces} array
715     *          argument or any of its elements are {@code null}, or
716     *          if the invocation handler, {@code h}, is
717     *          {@code null}
718     */
719    @CallerSensitive
720    public static Object newProxyInstance(ClassLoader loader,
721                                          Class<?>[] interfaces,
722                                          InvocationHandler h)
723        throws IllegalArgumentException
724    {
725        if (h == null) {
726            throw new NullPointerException();
727        }
728
729        final Class<?>[] intfs = interfaces.clone();
730        final SecurityManager sm = System.getSecurityManager();
731        if (sm != null) {
732            checkProxyAccess(Reflection.getCallerClass(), loader, intfs);
733        }
734
735        /*
736         * Look up or generate the designated proxy class.
737         */
738        Class<?> cl = getProxyClass0(loader, intfs);
739
740        /*
741         * Invoke its constructor with the designated invocation handler.
742         */
743        try {
744            final Constructor<?> cons = cl.getConstructor(constructorParams);
745            final InvocationHandler ih = h;
746            if (sm != null && ProxyAccessHelper.needsNewInstanceCheck(cl)) {
747                // create proxy instance with doPrivilege as the proxy class may
748                // implement non-public interfaces that requires a special permission
749                return AccessController.doPrivileged(new PrivilegedAction<Object>() {
750                    public Object run() {
751                        return newInstance(cons, ih);
752                    }
753                });
754            } else {
755                return newInstance(cons, ih);
756            }
757        } catch (NoSuchMethodException e) {
758            throw new InternalError(e.toString());
759        }
760    }
761
762    private static Object newInstance(Constructor<?> cons, InvocationHandler h) {
763        try {
764            return cons.newInstance(new Object[] {h} );
765        } catch (IllegalAccessException | InstantiationException e) {
766            throw new InternalError(e.toString());
767        } catch (InvocationTargetException e) {
768            Throwable t = e.getCause();
769            if (t instanceof RuntimeException) {
770                throw (RuntimeException) t;
771            } else {
772                throw new InternalError(t.toString());
773            }
774        }
775    }

777    /**
778     * Returns true if and only if the specified class was dynamically
779     * generated to be a proxy class using the {@code getProxyClass}
780     * method or the {@code newProxyInstance} method.
781     *
782     * <p>The reliability of this method is important for the ability
783     * to use it to make security decisions, so its implementation should
784     * not just test if the class in question extends {@code Proxy}.
785     *
786     * @param   cl the class to test
787     * @return  {@code true} if the class is a proxy class and
788     *          {@code false} otherwise
789     * @throws  NullPointerException if {@code cl} is {@code null}
790     */
791    public static boolean isProxyClass(Class<?> cl) {
792        return Proxy.class.isAssignableFrom(cl) && proxyClassCache.containsValue(cl);
793    }
794
795    /**
796     * Returns the invocation handler for the specified proxy instance.
797     *
798     * @param   proxy the proxy instance to return the invocation handler for
799     * @return  the invocation handler for the proxy instance
800     * @throws  IllegalArgumentException if the argument is not a
801     *          proxy instance
802     */
803    @CallerSensitive
804    public static InvocationHandler getInvocationHandler(Object proxy)
805        throws IllegalArgumentException
806    {
807        /*
808         * Verify that the object is actually a proxy instance.
809         */
810        if (!isProxyClass(proxy.getClass())) {
811            throw new IllegalArgumentException("not a proxy instance");
812        }
813
814        final Proxy p = (Proxy) proxy;
815        final InvocationHandler ih = p.h;
816        if (System.getSecurityManager() != null) {
817            Class<?> ihClass = ih.getClass();
818            Class<?> caller = Reflection.getCallerClass();
819            if (ReflectUtil.needsPackageAccessCheck(caller.getClassLoader(),
820                                                    ihClass.getClassLoader()))
821            {
822                ReflectUtil.checkPackageAccess(ihClass);
823            }
824        }
825
826        return ih;
827    }

037/**
038 * AccessibleContext represents the minimum information all accessible objects
039 * return.  This information includes the accessible name, description, role,
040 * and state of the object, as well as information about its parent and
041 * children.  AccessibleContext also contains methods for
042 * obtaining more specific accessibility information about a component.
043 * If the component supports them, these methods will return an object that
044 * implements one or more of the following interfaces:
045 * <P><ul>
046 * <li>{@link AccessibleAction} - the object can perform one or more actions.
047 * This interface provides the standard mechanism for an assistive
048 * technology to determine what those actions are and tell the object
049 * to perform them.  Any object that can be manipulated should
050 * support this interface.
051 * <li>{@link AccessibleComponent} - the object has a graphical representation.
052 * This interface provides the standard mechanism for an assistive
053 * technology to determine and set the graphical representation of the
054 * object.  Any object that is rendered on the screen should support
055 * this interface.
056 * <li>{@link  AccessibleSelection} - the object allows its children to be
057 * selected.  This interface provides the standard mechanism for an
058 * assistive technology to determine the currently selected children of the object
059 * as well as modify its selection set.  Any object that has children
060 * that can be selected should support this interface.
061 * <li>{@link AccessibleText} - the object presents editable textual information
062 * on the display.  This interface provides the standard mechanism for
063 * an assistive technology to access that text via its content, attributes,
064 * and spatial location.  Any object that contains editable text should
065 * support this interface.
066 * <li>{@link AccessibleValue} - the object supports a numerical value.  This
067 * interface provides the standard mechanism for an assistive technology
068 * to determine and set the current value of the object, as well as obtain its
069 * minimum and maximum values.  Any object that supports a numerical value
070 * should support this interface.</ul>
071 *
072 *
073 * @beaninfo
074 *   attribute: isContainer false
075 * description: Minimal information that all accessible objects return
076 *
077
078 * @author      Peter Korn
079 * @author      Hans Muller
080 * @author      Willie Walker
081 * @author      Lynn Monsanto
082 */

100                return accessibleContext.targetAppContext;
101            }

499     * Gets the role of this object.  The role of the object is the generic
500     * purpose or use of the class of this object.  For example, the role
501     * of a push button is AccessibleRole.PUSH_BUTTON.  The roles in
502     * AccessibleRole are provided so component developers can pick from
503     * a set of predefined roles.  This enables assistive technologies to
504     * provide a consistent interface to various tweaked subclasses of
505     * components (e.g., use AccessibleRole.PUSH_BUTTON for all components
506     * that act like a push button) as well as distinguish between sublasses
507     * that behave differently (e.g., AccessibleRole.CHECK_BOX for check boxes
508     * and AccessibleRole.RADIO_BUTTON for radio buttons).
509     * <p>Note that the AccessibleRole class is also extensible, so
510     * custom component developers can define their own AccessibleRole's
511     * if the set of predefined roles is inadequate.
512     *
513     * @return an instance of AccessibleRole describing the role of the object
514     * @see AccessibleRole
515     */

600     * Adds a PropertyChangeListener to the listener list.
601     * The listener is registered for all Accessible properties and will
602     * be called when those properties change.
603     *
604     * @see #ACCESSIBLE_NAME_PROPERTY
605     * @see #ACCESSIBLE_DESCRIPTION_PROPERTY
606     * @see #ACCESSIBLE_STATE_PROPERTY
607     * @see #ACCESSIBLE_VALUE_PROPERTY

611     *
612     * @param listener  The PropertyChangeListener to be added
613     */

615        if (accessibleChangeSupport == null) {
616            accessibleChangeSupport = new PropertyChangeSupport(this);
617        }
618        accessibleChangeSupport.addPropertyChangeListener(listener);
619    }

742     * is not empty, then fire a PropertyChange event to each listener.
743     * In general, this is for use by the Accessible objects themselves
744     * and should not be called by an application program.
745     * @param propertyName  The programmatic name of the property that
746     * was changed.
747     * @param oldValue  The old value of the property.
748     * @param newValue  The new value of the property.
749     * @see java.beans.PropertyChangeSupport
750     * @see #addPropertyChangeListener
751     * @see #removePropertyChangeListener
752     * @see #ACCESSIBLE_NAME_PROPERTY
753     * @see #ACCESSIBLE_DESCRIPTION_PROPERTY
754     * @see #ACCESSIBLE_STATE_PROPERTY
755     * @see #ACCESSIBLE_VALUE_PROPERTY
756     * @see #ACCESSIBLE_SELECTION_PROPERTY
757     * @see #ACCESSIBLE_TEXT_PROPERTY
758     * @see #ACCESSIBLE_VISIBLE_DATA_PROPERTY
759     */
760    public void firePropertyChange(String propertyName,
761                                   Object oldValue,
762                                   Object newValue) {
763        if (accessibleChangeSupport != null) {
764            if (newValue instanceof PropertyChangeEvent) {
765                PropertyChangeEvent pce = (PropertyChangeEvent)newValue;
766                accessibleChangeSupport.firePropertyChange(pce);
767            } else {
768                accessibleChangeSupport.firePropertyChange(propertyName,
769                                                           oldValue,
770                                                           newValue);
771            }
772        }
773    }

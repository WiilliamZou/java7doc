031/**

033 * that indicates whether the printer is currently able to accept jobs. This
034 * value is independent of the {@link PrinterState PrinterState} and {@link

038 * PrinterState PrinterState} is IDLE. If value is ACCEPTING_JOBS, the Printer
039 * will accept jobs even when the {@link PrinterState PrinterState} is STOPPED.
040 * <P>
